# homebrew-porsche-ev

Homebrew tap for the [porsche-ev](https://github.com/ejci/porsche-ev) CLI & TUI Dashboard.

## Installation

To install the `porsche-ev` command-line tool, run:

```bash
brew tap ejci/porsche-ev
brew install porsche-ev
```

## Usage

Once installed, you can start the dashboard with:

```bash
porsche-ev
```

For more options, run:

```bash
porsche-ev --help
```

## Requirements

- [Homebrew](https://brew.sh/)
- [Node.js](https://nodejs.org/) (will be installed automatically via Homebrew if not present)
